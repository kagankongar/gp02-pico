## Micropython Code on Raspberry Pico for Ai-Thinker GP-02 GPS Development Board

### Python code for GP-02 Development Board by Ai-Thinker
### No external libraries

#### Code includes basic functionality + pps

#### Connections
GP-02 TX  -> Pico 0 (phys 1)  
GP-02 RX  -> Pico 1 (phys 2)  
GP-02 PPS -> Pico 15 (phys 20)   
GP-02 GND -> Pico GND (phys 38)  
GP-02 VCC -> Pico 3.3V (phys 36)  

#### Build with Thonny (https://thonny.org/)
